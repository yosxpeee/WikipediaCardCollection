# Wikipedia Card Collection

## 概要
なんか巷で流行った某ゲームもどきのアプリです。  
面白半分で作ったので怒られたら消します()

## 動作環境
Windows 11 25H2以上  
python 3.11以上  

## 実行方法
### 一般ユーザ向け
releaseのところからzipファイルを拾ってお好きなところに展開してください。  
`WikipediaCardCollection.exe`をダブルクリックしてゲームを始めてください。

### プログラマ向け
コードをそのままpythonで実行するときは`start.bat`を実行してください。  
初回起動の場合各種インストールが行われてアプリが起動します。  
初回以降はインストールをスキップしてアプリが起動します。  
  
また、exeを作るときは`build.bat`を実行すると`dist`ディレクトリの下に`WikipediaCardCollection.exe`が生成されます。  

## ゲーム内容
### ガチャ
10連ガチャします。  
本家と違って10回ごとにSR以上、とかありません。強いのが欲しければガチ引きしてください。  
カードのステータスは本家と全然違います。  
また、素材カードが存在します(詳細は後述)。

### 図鑑
手に入れたカードのギャラリーみたいなもんです。  
各種フィルタやソート実装済み。お気に入り登録もできます。

### 強化
本家にはない要素です。  
素材としてカテゴライズされているカードを食わせることでランクを上げることができます。

### 模擬戦
手持ちのカード同士で1vs1のシングルマッチを行います。  
必ずプレイヤー側が先行、最大10ターンかけて交互に攻撃する簡単なオートバトルです。  
HPが0になったほうが負け、10ターン後も双方HPが残っていたらHPが多いほうが勝ちです。  
  
本家とはダメージの計算式が全然違います。  
まだクリティカルや回避といったものはありません。

### 設定
ダークモードとライトモードの切り替えができます。
