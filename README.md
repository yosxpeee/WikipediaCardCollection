## 動作環境
Windows 11 25H2以上  
python 3.11以上  

## 実行方法
### 一般ユーザ向け
releaseのところからzipファイルを拾ってお好きなところに展開してください。  
`WikipediaCardCollection.exe`をダブルクリックしてゲームを始めてください。

### プログラマ向け
コードをそのままpythonで実行するときは`start.bat`を実行してください。  
初回起動の場合各種インストールが行われてアプリが起動します。  
初回以降はインストールをスキップしてアプリが起動します。  
  
また、exeを作るときは`build.bat`を実行すると`dist`ディレクトリの下に`WikipediaCardCollection.exe`が生成されます。  

## ゲーム内容
### ガチャ
10連ガチャします。  
本家と違って10回ごとにSR以上、とかありません。強いのが欲しければガチ引きしてください。  
カードのステータスは本家と全然違います。  
また、素材カードが存在します(詳細は後述)。

### 図鑑
手に入れたカードのギャラリーみたいなもんです。  
各種フィルタやソート実装済み。お気に入り登録もできます。

### 強化
本家にはない要素です。  
素材としてカテゴライズされているカードを食わせることでランクを上げることができます。  
(LRをさらに強化はできません。あしからず。)  

### 模擬戦
手持ちのカード同士で1vs1のシングルマッチを行います。  
必ずプレイヤー側が先行、最大10ターンかけて交互に攻撃する簡単なオートバトルです。  
HPが0になったほうが負け、10ターン後も双方HPが残っていたら残HPの合計が多いほうが勝ちです。  
  
本家とはダメージの計算式が全然違います。ものすごいざっくりな計算です。
- 絶対値(防御側のDEF*(0.7～1.3のランダム)-攻撃側のATK)+200

をダメージとして防御側の残HPから差し引きます。
上記の計算で数値がマイナスにならなかった場合、割合ダメージとして
- (残HP*5%～10%)+100

という計算をしています。  
※割合ダメージでも一応相手を倒せるようにはしています。  

また、クリティカルやミスが存在します。
- クリティカル率：5%(相手へのダメージが1.5倍になる)
  - 割合ダメージで発生した場合は7.5～15%のダメージになる
- ミス率：3%(相手へのダメージが0になる)

### 出撃
6vs6の団体戦で専用エネミーが相手です。  
クリアごとに報酬があります(大したものではありません)。  
ダメージ計算は模擬戦と同一のものを使用します。  
| 難易度 | ランク |
| ------ | ---- |
| NORMAL | C級 |
| HARD | UC級 |
| VERY HARD | R級 |
| HARD CORE | SR級 |
| EXTREME | SSR級 |
| INSANE | UR級 |
| TORMENT | LR級 |
| LUNATIC | LR+級 |

### 設定
#### テーマ
ダークモードとライトモードの切り替えができます。
#### BGM
BGMの音量調整ができます。全BGM一括調整のみです。個別の音量調整には対応していません。  
また、各シーンごとのBGMをお好みのものに変更することができます。(mp3のみ対応)  
PC内のお好きなディレクトリにあるMP3ファイルをご指定ください。ループ再生して違和感のないものをオススメします。  

### 実績
Steamの実績っぽい何か  
達成すると画面下部に通知されます。
実績達成のチェックタイミングは以下の通りです。
- ガチャ：結果を閉じたとき
  - カードの保有状況ではなくその時に引いたカードを見るので、実績の実装前に達成したことがあっても後付けでの反映はされません。新たにもう一度達成してください。
- 出撃：報酬画面を閉じたとき
  - 実績実装前に達成していても後付けで反映されません。再度達成してください。
- 強化：強化の結果画面を閉じたとき
  - カードの保有状況を見るので、実績実装前に条件を満たしていれば1枚適当に強化した際に後付けで反映されます。
